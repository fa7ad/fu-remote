# fu-remote

Go based websocket server to receive commands from fu-remote app.

## INSTALL

**Method 1 (Go tool)**
---
Run
```bash
go get github.com/fa7ad/fu-remote
```

**Method 2 (binary)**
---
Download the latest binary build from [releases](https://github.com/fa7ad/fu-remote/releases)
